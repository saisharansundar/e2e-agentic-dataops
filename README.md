# E2E Agentic DataOps

## Project Goal
Build an end-to-end, data-agnostic, agentic data engineering system with AI agents that can:

1. Ingest messy datasets
2. Clean and transform data
3. Generate insights
4. Produce documentation automatically
5. Generate reports including slides

## Folder Structure
- `pipelines/` : ingestion and transformation scripts
- `agents/` : AI agents
- `tools/` : agent interfaces
- `data/` : raw, processed, and sample datasets
- `docs/` : generated documentation
- `outputs/` : reports and presentations
- `notebooks/` : exploration and experiments
